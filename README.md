# DERMATOLOGIST-CONSULTANCY-EXPERT-SYSTEM

## OBJECTIVE
Our python-based dermatology assist tool is an application that we hope to make it easier to figure out what might be going on with your skin and provide a suitable solution. 

## WORKING FEATURES
You can select your problem from a list of skin disease provided, then the model analyzes this information and gives you a list of possible solutions that you can then research further. You can also directly search for a treatment if you already know the problem with your skin. We will regularly update the information about current skin diseases and as well as new diseases and its treatments to provide better results.

## PROGRAMMING LANGUAGE
Python (Jupyter or PyCharm)
